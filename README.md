UPI Fraud Detection using Machine Learning

This project builds an end-to-end machine learning pipeline to detect fraudulent UPI transactions in digital payment systems. With the rapid growth of digital payments in India, detecting suspicious transactions is critical for financial security.

The system analyzes transaction features such as transaction amount, transaction time, device type, account age, and location changes to predict whether a transaction is fraudulent or legitimate.

**Objectives**

1. Build a fraud detection model using machine learning

2. Handle financial transaction data

3. Train and evaluate classification models

4. Deploy the model as a REST API for real-time prediction

**Technologies Used**

Python

Pandas & NumPy

Scikit-learn

Flask API
